Настройка ES6

  • Открыть папку в cmd, нажать сочетание клавиш windows+r, написать cmd, написать в строчке cd и путь к папке, куда необходимо установить babel;
  • npm init -y (согласиться с требованиями);
  • npm install babel-cli babel-core babel-preset-es2015 --save-dev (установка babel);
    • babel-cli - интерфейс командной строки;
    • babel-core - основной модуль babel;
    • babel-preset-es2015 - модуль необходимый для транспиляции ES6;
    • --save-dev - добавит файл package.json в котором указываются модули для разработки;
  • Cоздать папки src и dist;
    • src - начальный код;
    • dist - конечный код;
  • В файле package.json добавить :
    • "build": "babel src -d dist --presets es2015", "watch": "babel src -d dist --presets es2015 -w"
  • Написать в cmd : npm run watch - авто компиляция.
