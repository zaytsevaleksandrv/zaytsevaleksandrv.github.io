Настройка React.js

  • Открыть папку в cmd, нажать сочетание клавиш windows+r, написать cmd, написать в строчке cd и путь к папке, куда необходимо установить babel;
  • cmd => cd <'name папка'>
  • cmd => npm init
  • cmd => npm install react
  • cmd => npm install <после заполенния package.json>
  • new folder -> <'src, public'>.
  • <'src'> create new file <'index.js'>
  • cmd => npm i webpack --save-dev
  • create new folder in folder <'public'> -> <'build'>
  • create new file in folder <'build'> -> index.html
  • cmd => npm install webpack -g
  • cmd=> webpack
  • cmd=> npm i json-loader --save-dev
  • cmd=> npm run devserver
